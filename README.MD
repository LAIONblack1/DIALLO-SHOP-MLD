# Diallo Shop MLD

E-commerce complet avec Next.js, Tailwind CSS, Context API.

## Fonctionnalités

- Catalogue produits avec filtres (catégorie, prix, recherche)
- Panier / Wishlist (localStorage)
- Authentification mock (JWT simulé)
- Dashboard utilisateur (commandes, infos)
- Interface admin (ajout/modif/suppression produit, gestion commandes)
- WhatsApp flottant, notifications toast, scroll to top
- Responsive design

## Installation

```bash
npm install
npm run dev
